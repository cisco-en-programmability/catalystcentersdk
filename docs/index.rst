============
catalystcentersdk
============

*Simple, lightweight, scalable Python API wrapper for the CatalystCenter APIs*

-------------------------------------------------------------------------------

Welcome to the docs!  catalystcentersdk is a *community developed* Pythonic wrapping of the CatalystCenter APIs (for API versions:  2.2.2.3, 2.2.3.3, 2.3.3.0, 2.3.5.3 and 2.3.7.6).
The package represents all of the Cisco CatalystCenter API interactions via native Python tools.
Making working with the Cisco CatalystCenter APIs in Python a *native* and *natural* experience.

**catalystcentersdk helps you get things done faster.**  We take care of the API semantics, and you can focus on writing your code.

With catalystcentersdk, you can easily:

* Interact with the CatalystCenter APIs in an interactive Python session

* Quickly create code that enables you get something done in CatalystCenter

* Leverage the API wrapper to cleanly add CatalystCenter functionality to your project without having to write the boilerplate code for working with the CatalystCenter APIs

To *dive in* and see how catalystcentersdk makes your life better, check out the
:ref:`Quickstart`!
